Visão Geral do SGC-Saúde

O SGC-Saúde (Sistema de Gestão de Contratos) é uma aplicação web desenvolvida em Python com o framework Flask, projetada para ser uma base para a gestão de Atas de Registro de Preço, Contratos, Empenhos e o consumo de itens/serviços vinculados. O sistema organiza e centraliza o controle de documentos e o fluxo de gastos, incluindo um sistema de cotas por unidade de saúde para um acompanhamento detalhado.
Este sistema foi desenvolvido como um projeto piloto para a Secretaria de Saúde de Nova Friburgo.

Como Usar o Sistema

1. Ponto de Partida
Para começar, é essencial que os dados básicos estejam cadastrados. Inicie cadastrando as Unidades de Saúde e os Fornecedores, pois eles serão necessários ao criar os demais documentos.
2. Entendendo os Documentos
  • - Contratos: São documentos independentes, com objeto, valor e itens próprios. Podem ter sua vigência e valor alterados por meio de Termos Aditivos.
  • - Contratinhos e Empenhos: São documentos que servem para registrar o consumo de itens de uma Ata de Registro de Preço já existente, debitando o saldo do item e da cota da unidade solicitante.
